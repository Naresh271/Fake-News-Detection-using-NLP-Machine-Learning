🔹 Fake News Detection using NLP & Machine Learning
📌 Project Overview

This project aims to classify news statements as real or fake using Natural Language Processing and Machine Learning techniques.

🛠️ Technologies Used
Python
Pandas, NumPy
Scikit-learn
TF-IDF Vectorization
⚙️ Workflow
Data Collection (train & test datasets)
Data Cleaning and Preprocessing
Text Normalization
Feature Extraction using TF-IDF
Model Training using Logistic Regression
Model Evaluation
📊 Model Performance
Accuracy: 85%
Evaluated using classification metrics
🚀 Features
Predicts whether news is Real or Fake
Handles text preprocessing automatically
Custom prediction function for new inputs
